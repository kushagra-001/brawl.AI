---

## 🔄 Processing Flow

1. **Upload** — JPEG, PNG, WEBP · max 5MB
2. **Validate** — File type & size check
3. **Game Initialization** — Mode selection (AI / PvP / Multiplayer)
4. **Matchmaking** — Player or AI opponent assigned
5. **Battle Execution** — Real-time actions & decisions
6. **Score Calculation** — Logic + performance evaluation
7. **Result Generation** — Winner, score, breakdown
8. **DB Logging** — Match data saved to MongoDB
9. **Response** — Result returned to frontend

## 🧠 Score Calculation System

The scoring engine evaluates player performance based on multiple factors such as strategy, speed, and accuracy.

The scoring system works as follows:

```id="scoreblock1"
Base Score: 50  (neutral starting point)

        ┌───────────────────────────────┐
        │       SCORE BREAKDOWN         │
        ├───────────────────────────────┤
        │ Factor            | Impact    │
        ├───────────────────────────────┤
        │ Strategy Score    | +30       │
        │ Speed Score       | +20       │
        │ Accuracy Score    | +25       │
        │ Mistakes Penalty  | -15       │
        │ Combo Bonus       | +10       │
        ├───────────────────────────────┤
        │ Final Score       | 0 - 100   │
        └───────────────────────────────┘
```

* Scoring adapts based on player decisions and gameplay style
* AI difficulty influences score variation
* Final score determines match outcome and ranking

## 📡 API Endpoints

### 🎮 Start Match

`POST /api/start-match`

Start a new game session.

#### Request

```
Body    : JSON
{
  "mode": "AI", 
  "player_id": "user123"
}
```

#### Response

```json
{
  "match_id": "abc123",
  "status": "started"
}
```

---

### ⚔️ Submit Move

`POST /api/move`

Send player action during gameplay.

#### Request

```
Body    : JSON
{
  "match_id": "abc123",
  "move": "attack"
}
```

#### Response

```json
{
  "result": "success",
  "next_turn": "AI"
}
```

---

### 🏆 Get Match Result

`GET /api/result`

Fetch final result of a match.

#### Response

```json
{
  "match_id": "abc123",
  "winner": "player",
  "score": 92,
  "confidence": "High",
  "breakdown": {
    "strategy_score": 80,
    "speed_score": 95,
    "accuracy_score": 90
  },
  "timestamp": "2026-04-02T10:30:00Z"
}
```

---

### 📊 Get Match History

`GET /api/history`

Fetch recent matches from database.

#### Response

```json
{
  "results": [
    "...array of past matches..."
  ]
}
```
